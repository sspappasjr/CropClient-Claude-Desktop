# CropClient Irrigation Glossary

## Purpose

This glossary defines the standard irrigation vocabulary used by ClientAI.  
ClientAI must use these meanings consistently and must not invent alternative interpretations.

---

### Ranch
Top-level geographic or operational unit containing one or more blocks.

### Block
Sub-unit within a ranch representing a specific planted area.

### Irrigation Event
A recorded instance of water applied to a block, including date, duration, and method.

### Recommendation (Crown Jewel)
System-generated irrigation guidance based on crop stage, ET, soil, and prior irrigation.
