### Napkin: Page Designer v1.1 Specs

**Big Picture:**
- Designer = section locator. Load a real app, design its sections (S1-S4). Not building the whole app.
- No HEADER component — app already has its own header. Only S1-S4 designable.
- JSON design saves/loads right back where tools were. Genius.
- Designer can add injector markers to apps that don't have them — same skill as Factory.
- Finish design work first, coding discussion comes later.

**Layout (2 rows):**
```
Row 1: [folder] | [filename] | "Factory Layout Designer" | [SAVE] [SAVE AS] [☑ JSON Too]
Row 2: [OPEN] | [icons...] [Same Size] [Grp All] | iPad ▼ | Scroll ▼ | Cols/Rows % | CLEAR | Injectors [n] All [n] | >>>
```
- Row 2 scrollable with horizontal slide bar
- Saves a line vs current 3-row layout

**Save/Open:**
- SAVE = quick save (no dialog if already saved), first time = acts like SAVE AS
- SAVE AS = always opens explorer
- JSON Too checkbox (default ON) = on save, auto-saves per-section JSON
- OPEN = remembers last folder, loads JSON right back
- All operations remember last folder — no more hunting

**JSON Per Section:**
- Saves separate files: -S1.json, -S2.json, -S3.json, -S4.json
- Only sections with tools get saved
- Naming follows Steve's pattern: Factory-MCP-2x2-Designer-Admin-v1.0-S1.json
- Section detected by tool position (logic already in code)

**Injector Controls (next to CLEAR):**
- No EXPORT button — replaced by injector controls
- Injectors [count] = how many @@@@ markers exist
- All [count] = total sections that should have markers
- All button = auto-add markers for all sections
- Designer or Factory can install injectors — same skill

**Same Size Icon:**
- Tool bar icon. Size one tool → copy 5x → select all → Same Size → all match
- No pulling that distorts font/perspective

**Group All Icon:**
- Tool bar icon. One click = select all + group
- Toggle ON/OFF. Nothing moves on toggle.
- ON + drag = everything moves together

**Designer Resource (future):**
- Tool palette saved as resource file with standard name
- Page Builder injects it: "Add Designer" → save → it's a page designer
